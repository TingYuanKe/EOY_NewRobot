# EOY-Robot Project

## EyeOnYouDepthCamera

### Keyword command

* s : Draw skeleton
* l : Draw user status label
* c : Draw center of mass
* x : Draw bounding box
* b : Draw background
* d :  Draw depth
* f : Draw frame ID
* r : Run Robot Tracking
* l : Draw user status label
* 1: draw  both RGB abd depth scene
* 2: draw only depth scene
* 3: draw only RGB scene
